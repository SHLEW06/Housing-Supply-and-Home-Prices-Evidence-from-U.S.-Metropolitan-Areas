ECON320 LAB Final Project — Brief Rubric

Core things I’m looking for (weights):

Rubric
  1. Motivation & question (25%) Why this matters; a clear causal/economic question.
  2. Design & data (25%) Identification strategy fits the question; data choice is sensible.
  3. Analysis & execution (15%) Clean workflow; correct methods (e.g., t/F tests, robust SEs); code runs.
  4. Interpretation & insight (25%) Results read in plain language; effect sizes and uncertainty; limits & robustness noted.
  5. Clarity & professionalism (10%) Well-organized writeup; figures/tables labeled; reproducible code.
Total = 100%.
